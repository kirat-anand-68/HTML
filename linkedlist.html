Introduction to Linked Lists

A linked list is a linear data structure where each element (node) contains a reference to the next node in the sequence.

Types of Linked Lists

  • Singly Linked List: Each node point to the next node.
  • Doubly Linked List Each node point to both the next and previous nodes.
  • Circular Linked List: The last node links back to the first node

Linked List Examples

Python

            class Node:
                def __init__(self, data):
                    self.data = data
                    self.next = None
            
            head = Node(1)
            head.next = Node(2)
            print(head.data)  # Output: 1

Java

            class Node {
                int data;
                Node next;
                Node(int data) {
                    this.data = data;
                    this.next = null;
                }
            }

C++

            class Node {
            public:
                int data;
                Node* next;
                Node(int data) {
                    this->data = data;
                    this->next = nullptr;
                }
            };
