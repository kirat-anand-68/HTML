Introduction to Stacks

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle. Elements are added and removed from the same end, called the top of the stack.

Types of Arrays

  • Push: Adds an Element to the top of the Stack
  • PoP: Removes the top Element from the Stack .
  • is Empty: Checks if the Stack is Empty.

Stack Examples

Python

            stack = []
            stack.append(1)  # Push
            stack.append(2)
            print(stack.pop())  # Pop: Output 2

Java

            import java.util.Stack;
            Stack stack = new Stack<>();
            stack.push(1);
            stack.push(2);
            System.out.println(stack.pop()); // Output: 2

C++

            #include 
            std::stack stack;
            stack.push(1);
            stack.push(2);
            std::cout << stack.top(); // Output: 2
            stack.pop();
