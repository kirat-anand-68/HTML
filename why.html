Algorithm Steps:

1. Start
2. Get the number of rows and columns for the matrices (say, matrix A and matrix B).
3. Accept the elements for both matrices A and B.
4.Ensure that both matrices have the same dimensions (same number of rows and columns). If they do not have the same dimensions, matrix addition is not possible.
5. Initialize an empty matrix C with the same dimensions as matrices A and B.
6. For each element in the matrices, add the corresponding elements of matrix A and matrix B and store the result in matrix C. For every element C[i][j] = A[i][j] + B[i][j], where i and j are the row and column indices respectively.
7. Print the resulting matrix C.
8.End

Code :

                                #include <iostream>
                                using namespace std;

                                    int a[rows][cols], b[rows][cols], sum[rows][cols];
                                
                                    // Input for first matrix
                                    cout << "Enter elements of first matrix:\n";

Q. Design, Develop and Implement a menu driven Program for the following operations on STACK of Integers (Array Implementation of Stack with maximum size MAX)
a. Push an Element on to Stack
b. Pop an Element from Stack
c. To Check Overflow and Underflow situations on Stack
d. Display

Algorithm Steps for adding two matrices:


1.Initialize the Stack:
Define an array stack[MAX] where MAX is the maximum size of the stack.
Set the top = -1, indicating that the stack is initially empty.
2.Check for Overflow:
If top == MAX - 1, then the stack is full and Overflow occurs.
3.Check for Underflow:
If top == -1, the stack is empty and Underflow occurs.
4.Push Operation:
If the stack is not full (i.e., top != MAX - 1):
Increment top by 1.
Push the new element element onto stack[top].
5.Pop Operation:
If the stack is not empty (i.e., top != -1):
Pop the element from stack[top] and store it in a variable (optional).
Decrement top by 1.
6.Display Stack:
Print all elements from stack[0] to stack[top] to display the stack elements

Code :

                                #include <iostream>
                                #include <cstring>
                                using namespace std;
                                
                                #define MAX 5  // Maximum size of the stack
                                
                                class Stack {
                                private:
                                    int stack[MAX];   // Array to hold the stack elements
                                    int top;          // Index for the top of the stack
                                
                                public:
                                    // Constructor to initialize the stack
                                    Stack() {
                                        top = -1;  // Stack is initially empty
                                    }
                                
                                    // Function to check if the stack is full
                                    bool isFull() {
                                        return top == MAX - 1;
                                    }
                                
                                    // Function to check if the stack is empty
                                    bool isEmpty() {
                                        return top == -1;
                                    }
                                
                                    // Function to push an element onto the stack
                                    void push(int element) {
                                        if (isFull()) {
                                            cout << "Overflow! Stack is full. Cannot push " << element << endl;
                                        } else {
                                            stack[++top] = element;
                                            cout << element << " pushed onto stack." << endl;
                                        }
                                    }
                                
                                    // Function to pop an element from the stack
                                    void pop() {
                                        if (isEmpty()) {
                                            cout << "Underflow! Stack is empty. Cannot pop an element." << endl;
                                        } else {
                                            cout << "Popped element: " << stack[top--] << endl;
                                        }
                                    }
                                
                                    // Function to display the stack
                                    void display() {
                                        if (isEmpty()) {
                                            cout << "Stack is empty." << endl;
                                        } else {
                                            cout << "Stack elements (top to bottom): ";
                                            for (int i = top; i >= 0; i--) {
                                                cout << stack[i] << " ";
                                            }
                                            cout << endl;
                                        }
                                    }
                                
                                    // Function to check if the sequence is a palindrome
                                    void checkPalindrome() {
                                        if (isEmpty()) {
                                            cout << "Stack is empty. Cannot check for palindrome." << endl;
                                            return;
                                        }
                                
                                        int size = top + 1;
                                        int *temp = new int[size]; // Temporary array to reverse the stack
                                        int index = 0;
                                
                                        // Copy stack elements into the temporary array
                                        for (int i = top; i >= 0; i--) {
                                            temp[index++] = stack[i];
                                        }
                                
                                        bool isPalindrome = true;
                                        for (int i = 0; i < size; i++) {
                                            if (stack[i] != temp[i]) {
                                                isPalindrome = false;
                                                break;
                                            }
                                        }
                                
                                        if (isPalindrome) {
                                            cout << "The stack represents a palindrome sequence." << endl;
                                        } else {
                                            cout << "The stack does not represent a palindrome sequence." << endl;
                                        }
                                
                                        delete[] temp;
                                    }
                                };
                                
                                int main() {
                                    Stack stack;
                                    int choice, element;
                                    
                                    do {
                                        // Display menu options
                                        cout << "\nMenu:\n";
                                        cout << "1. Push an Element onto Stack\n";
                                        cout << "2. Pop an Element from Stack\n";
                                        cout << "3. Check Palindrome using Stack\n";
                                        cout << "4. Display Stack\n";
                                        cout << "5. Exit\n";
                                        cout << "Enter your choice: ";
                                        cin >> choice;
                                
                                        switch (choice) {
                                            case 1: // Push an Element
                                                cout << "Enter element to push: ";
                                                cin >> element;
                                                stack.push(element);
                                                break;
                                
                                            case 2: // Pop an Element
                                                stack.pop();
                                                break;
                                
                                            case 3: // Check Palindrome
                                                stack.checkPalindrome();
                                                break;
                                
                                            case 4: // Display Stack
                                                stack.display();
                                                break;
                                
                                            case 5: // Exit
                                                cout << "Exiting program...\n";
                                                break;
                                
                                            default:
                                                cout << "Invalid choice! Please try again.\n";
                                        }
                                    } while (choice != 5);
                                
                                    return 0;
                                }

Q. Design, Develop and Implement a Program for converting an Infix Expression to Postfix Expression. Program should support for both parenthesized and free parenthesized expressions with the operators: +, -, *, /, %(Remainder), ^(Power) and alphanumeric operands

Algorithm Steps :


1.Initialize Two Data Structures:
Stack for operators and parentheses. Result (Postfix Expression) to store the final postfix expression.
2.Process Each Character in the Infix Expression:
If the character is an operand (alphanumeric):
Append it directly to the result (Postfix expression).
If the character is a left parenthesis (:
Push it onto the stack.
If the character is a right parenthesis ):
Pop from the stack and append to the result until a left parenthesis( \is encountered. Discard the left parenthesis.
If the character is an operator (+, -, *, /, %, ^):
While the stack is not empty and the top of the stack has an operator of greater or equal precedence, pop operators from the stack and append them to the result.
Push the current operator onto the stack.
After Scanning the Infix Expression:
Pop any remaining operators from the stack and append them to the result.
3.Return the Postfix Expression:
The result now contains the postfix expression.
Operator Precedence and Associativity:
Operators + and - have the lowest precedence.
Operators *, /, and % have higher precedence than + and -.
Operator ^ (Power) has the highest precedence.
+, -, *, /, and % are left-associative.
^ is right-associative

Code :

                                #include <iostream>
                                #include <stack>
                                #include <cctype>  // For isalnum()
                                using namespace std;
                                
                                class InfixToPostfix {
                                private:
                                    stack<char> opStack;  // Stack to hold operators
                                    string postfix;       // String to store postfix expression
                                
                                    // Function to return precedence of operators
                                    int precedence(char op) {
                                        if (op == '^') return 3;
                                        if (op == '*' || op == '/' || op == '%') return 2;
                                        if (op == '+' || op == '-') return 1;
                                        return 0;
                                    }
                                
                                    // Function to check if a character is an operator
                                    bool isOperator(char c) {
                                        return (c == '+' || c == '-' || c == '*' || c == '/' || c == '%' || c == '^');
                                    }
                                
                                public:
                                    // Function to convert infix to postfix
                                    string convert(string infix) {
                                        for (char ch : infix) {
                                            if (isalnum(ch)) {  // If operand, add to postfix
                                                postfix += ch;
                                            } 
                                            else if (ch == '(') {  // If '(', push to stack
                                                opStack.push(ch);
                                            } 
                                            else if (ch == ')') {  // If ')', pop till '('
                                                while (!opStack.empty() && opStack.top() != '(') {
                                                    postfix += opStack.top();
                                                    opStack.pop();
                                                }
                                                opStack.pop();  // Remove '('
                                            } 
                                            else if (isOperator(ch)) {  // If operator
                                                while (!opStack.empty() && precedence(opStack.top()) >= precedence(ch)) {
                                                    postfix += opStack.top();
                                                    opStack.pop();
                                                }
                                                opStack.push(ch);
                                            }
                                        }
                                
                                        // Pop remaining operators
                                        while (!opStack.empty()) {
                                            postfix += opStack.top();
                                            opStack.pop();
                                        }
                                
                                        return postfix;
                                    }
                                
                                    // Function to display postfix expression
                                    void display() {
                                        cout << "Postfix Expression: " << postfix << endl;
                                    }
                                };
                                
                                // Driver code
                                int main() {
                                
                                    string infix;
                                    cout << "Enter an infix expression: ";
                                    cin >> infix;
                                
                                    InfixToPostfix converter;
                                    string postfix = converter.convert(infix);
                                    converter.display();
                                
                                    return 0;
                                }

Q. Design, Develop and Implement a Program for evaluating Suffix expression with single digit operands and operators: +, -, *, /, %, ^.

Algorithm Steps :


1.Initialize an Empty Stack:
Create an empty stack to store operands.
2.Process Each Character of the Suffix Expression:
Push the operand (single-digit number) onto the stack.Pop the top two operands from the stack. Apply the operator on the operands. Push the result back onto the stack.
3.End of Expression:
After processing the entire expression, the stack will contain only one value, which is the result of the expression.
4.Return the Final Result:
Pop and return the final result from the stack

Code :

                                #include <iostream>
                                #include <stack>
                                #include <cmath>  // For power function
                                
                                using namespace std;
                                
                                // Function to evaluate a postfix expression
                                int evaluatePostfix(string expression) {
                                    stack<int> s;
                                
                                    for (char ch : expression) {
                                        // If character is an operand (digit), push it to stack
                                        if (isdigit(ch)) {
                                            s.push(ch - '0');  // Convert character to integer
                                        } 
                                        else {
                                            // Pop two operands
                                            int operand2 = s.top(); s.pop();
                                            int operand1 = s.top(); s.pop();
                                            int result;
                                
                                            // Perform the operation
                                            switch (ch) {
                                                case '+': result = operand1 + operand2; break;
                                                case '-': result = operand1 - operand2; break;
                                                case '*': result = operand1 * operand2; break;
                                                case '/': result = operand1 / operand2; break;
                                                case '%': result = operand1 % operand2; break;
                                                case '^': result = pow(operand1, operand2); break;
                                                default: cout << "Invalid operator encountered!" << endl; return -1;
                                            }
                                
                                            // Push the result back onto the stack
                                            s.push(result);
                                        }
                                    }
                                
                                    // The final result is in the stack
                                    return s.top();
                                }
                                
                                // Main function
                                int main() {
                                
                                    string postfixExpression;
                                    
                                    // Input: Postfix expression from the user
                                    cout << "Enter a valid postfix expression: ";
                                    cin >> postfixExpression;
                                
                                    // Evaluate the expression
                                    int result = evaluatePostfix(postfixExpression);
                                    
                                    // Output the result
                                    cout << "Result: " << result << endl;
                                
                                    return 0;
                                }

Q. Design, Develop and Implement a Program for solving Tower of Hanoi problem with n disks

Algorithm Steps :


1.Base Case:
If n == 1, move the disk from source peg to destination peg.
2.Recursive Case:
If n > 1, do the following:
Move n-1 disks from source peg to auxiliary peg using the destination peg as temporary storage. Move the nth disk from source peg to destination peg. Move the n-1 disks from auxiliary peg to destination peg using the source peg as temporary storage.

Code :

                                #include <iostream>
                                using namespace std;
                                
                                // Recursive function to solve Tower of Hanoi
                                void towerOfHanoi(int n, char source, char auxiliary, char destination) {
                                    if (n == 1) {
                                        // Base case: Move the smallest disk directly
                                        cout << "Move disk 1 from " << source << " to " << destination << endl;
                                        return;
                                    }
                                
                                    // Step 1: Move n-1 disks from source to auxiliary
                                    towerOfHanoi(n - 1, source, destination, auxiliary);
                                
                                    // Step 2: Move nth disk from source to destination
                                    cout << "Move disk " << n << " from " << source << " to " << destination << endl;
                                
                                    // Step 3: Move n-1 disks from auxiliary to destination
                                    towerOfHanoi(n - 1, auxiliary, source, destination);
                                }
                                
                                // Main function
                                int main() {
                                
                                    int n;
                                
                                    // Taking input: number of disks
                                    cout << "Enter the number of disks: ";
                                    cin >> n;
                                
                                    // Solving Tower of Hanoi
                                    cout << "Steps to solve Tower of Hanoi:\n";
                                    towerOfHanoi(n, 'A', 'B', 'C');
                                
                                    return 0;
                                }

Q. Design, Develop and Implement a menu driven Program for the following operations on Linear QUEUE (Array Implementation of Queue with maximum size MAX)
a. Check Overflow and Underflow situations on Linear QUEUE.
b. Insert an Element on to Linear QUEUE.
c. Delete an Element from Linear QUEUE.
d. Display the elements of Linear QUEUE.
e. Exit.
Support the program with appropriate functions for each of the above operations .

Algorithm Steps :


1. Initialization:
Declare a queue array with a maximum size MAX.Initialize front = -1 and rear = -1 to represent an empty queue.
2.Check Overflow:
The queue is full if rear == MAX - 1.
3.Check Underflow:
The queue is empty if front == -1 and rear == -1.
4.Insert an Element (Enqueue):
If rear == MAX - 1, display overflow. Otherwise, increment rear and insert the element at the rear position.If front == -1, set front = 0 as the first element is inserted.
5.Delete an Element (Dequeue):
If front == -1 (empty queue), display underflow. Otherwise, remove the element at the front position and increment front. If front > rear after deletion, reset front and rear to -1 (empty queue).
6.Display the Elements:
If the queue is not empty, iterate from front to rear and print the elements. If the queue is empty, display an appropriate message.
7.Exit

Code :

                                #include <iostream>
                                #define MAX 5  // Maximum size of the queue
                                
                                using namespace std;
                                
                                class Queue {
                                private:
                                    int arr[MAX];  // Array to store queue elements
                                    int front, rear;
                                
                                public:
                                    // Constructor to initialize the queue
                                    Queue() {
                                        front = -1;
                                        rear = -1;
                                    }
                                
                                    // Check if the queue is full (Overflow condition)
                                    bool isFull() {
                                        return (rear == MAX - 1);
                                    }
                                
                                    // Check if the queue is empty (Underflow condition)
                                    bool isEmpty() {
                                        return (front == -1 || front > rear);
                                    }
                                
                                    // Insert an element into the queue (Enqueue)
                                    void enqueue(int value) {
                                        if (isFull()) {
                                            cout << "Queue Overflow! Cannot insert " << value << endl;
                                            return;
                                        }
                                        if (front == -1) front = 0;  // Set front to 0 if inserting the first element
                                        arr[++rear] = value;
                                        cout << "Inserted: " << value << endl;
                                    }
                                
                                    // Remove an element from the queue (Dequeue)
                                    void dequeue() {
                                        if (isEmpty()) {
                                            cout << "Queue Underflow! No elements to delete.\n";
                                            return;
                                        }
                                        cout << "Deleted: " << arr[front] << endl;
                                        front++;
                                    }
                                
                                    // Display the elements in the queue
                                    void display() {
                                        if (isEmpty()) {
                                            cout << "Queue is empty!\n";
                                            return;
                                        }
                                        cout << "Queue elements: ";
                                        for (int i = front; i <= rear; i++)
                                            cout << arr[i] << " ";
                                        cout << endl;
                                    }
                                };
                                
                                // Main function with menu-driven operations
                                int main() {
                                
                                    Queue q;
                                    int choice, value;
                                
                                    do {
                                        cout << "\nQueue Operations Menu:\n";
                                        cout << "1. Insert (Enqueue)\n";
                                        cout << "2. Delete (Dequeue)\n";
                                        cout << "3. Display Queue\n";
                                        cout << "4. Exit\n";
                                        cout << "Enter your choice: ";
                                        cin >> choice;
                                
                                        switch (choice) {
                                            case 1:
                                                cout << "Enter value to insert: ";
                                                cin >> value;
                                                q.enqueue(value);
                                                break;
                                            case 2:
                                                q.dequeue();
                                                break;
                                            case 3:
                                                q.display();
                                                break;
                                            case 4:
                                                cout << "Exiting program...\n";
                                                break;
                                            default:
                                                cout << "Invalid choice! Try again.\n";
                                        }
                                    } while (choice != 4);
                                
                                    return 0;
                                }
