Introduction to Arrays

An array is a data structure that stores a collection of elements of the same type. Arrays allow for efficient indexing and manipulation of data.

Types of Arrays

  • One-dimensional array: A simple list of elements.
  • Multi-dimensional array: An array with more than one level (e.g., a matrix).
  • Dynamic array: An array that can resize dynamically.

Array Examples

Python

# Creating an array in Python
arr = [1, 2, 3, 4, 5]
print(arr[0])  # Accessing elements

Java

// Creating an array in Java
int[] arr = {1, 2, 3, 4, 5};
System.out.println(arr[0]); // Accessing elements

C++

// Creating an array in C++
int arr[] = {1, 2, 3, 4, 5};
std::cout << arr[0]; // Accessing elements
