Introduction to Heaps

A heap is a specialized tree-based data structure that satisfies the heap property. It is commonly used to implement priority queues.

Types of Heaps

  • Min Heap: The parent node is always smaller than or equal to its child nodes.
  • Max Heap: The parent node is always greater than or equal to its child nodes.

Heap Examples

Python

            import heapq
            heap = []
            heapq.heappush(heap, 3)
            heapq.heappush(heap, 1)
            heapq.heappush(heap, 2)
            print(heapq.heappop(heap))  # Output: 1 (Min Heap)

Java

            import java.util.PriorityQueue;
            PriorityQueue heap = new PriorityQueue<>();
            heap.add(3);
            heap.add(1);
            heap.add(2);
            System.out.println(heap.poll()); // Output: 1 (Min Heap)
