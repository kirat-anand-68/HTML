Introduction to Disjoint Sets

Disjoint Set (or Union-Find) is a data structure that keeps track of a partition of a set into disjoint subsets. It is useful in applications like Kruskalâ€™s algorithm for Minimum Spanning Tree.

Operations in Disjoint Set

  • Find: Determines which set a particular element is in.
  • Union: Merges two sets together.
  • Path Compression: Optimizes the Find operation.

Disjoint Set Examples

Python

            class DisjointSet:
                def __init__(self, n):
                    self.parent = list(range(n))
                
                def find(self, x):
                    if self.parent[x] != x:
                        self.parent[x] = self.find(self.parent[x])
                    return self.parent[x]
                
                def union(self, x, y):
                    rootX = self.find(x)
                    rootY = self.find(y)
                    if rootX != rootY:
                        self.parent[rootY] = rootX

Java

            class DisjointSet {
                int[] parent;
                DisjointSet(int n) {
                    parent = new int[n];
                    for (int i = 0; i < n; i++)
                        parent[i] = i;
                }
                int find(int x) {
                    if (parent[x] != x)
                        parent[x] = find(parent[x]);
                    return parent[x];
                }
                void union(int x, int y) {
                    parent[find(y)] = find(x);
                }
            }
